## Add-on Name
BiliShelf Helper

## Summary
Capture Bilibili video metadata and save it to your local manager in one click.

## Description
BiliShelf Helper is built for Bilibili video collection management, including capture, folder assignment, and tag organization.

Features:
- Floating capture panel on video pages
- Automatic metadata extraction
- Multi-folder assignment
- Custom tag input
- Theme switching

## Reviewer Notes
- Does not collect credentials
- Does not inject ads or tracking scripts
- Requests only required domains (Bilibili pages/APIs and local backend)
