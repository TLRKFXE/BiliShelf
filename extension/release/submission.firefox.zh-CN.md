# Firefox AMO 提交文案（中文可直接粘贴）

## Add-on 名称
BiliShelf Helper

## 摘要
在 B 站视频页一键采集并保存视频信息到本地管理系统。

## 详细说明
BiliShelf Helper 用于 Bilibili 视频收藏管理，支持视频信息采集、收藏夹归档和标签管理。

功能包括：
- 视频页悬浮采集面板
- 自动读取视频元信息
- 多收藏夹归档
- 自定义标签
- 主题切换

## Firefox 审核补充说明（可放备注）
- 不收集登录凭据
- 不注入广告或跟踪脚本
- 仅在必要域名请求数据（Bilibili 页面与 API、本地后端）
