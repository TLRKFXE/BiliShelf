## Detailed Description
BiliShelf Helper helps you turn scattered Bilibili videos into a searchable, structured local collection.

Key features:
1. Floating capture panel on Bilibili video pages
2. Auto-read metadata: BV, title, uploader, cover, description, system tags
3. Save videos into multiple folders with custom tags
4. Integrates with your local manager backend API
5. Light / Dark / Auto theme support

Notes:
- Active only on `bilibili.com/video/*`
- Data is saved through your local backend service
- No cloud account system is included
