## 应用名称
BiliShelf Helper

## 简短描述（建议 ≤ 132 字符）
在 B 站视频页一键采集视频信息，快速保存到本地管理系统，支持多收藏夹与标签管理。

## 详细描述（可直接使用）
BiliShelf Helper 是一个面向 Bilibili 视频收藏管理的效率工具，帮助你把零散视频沉淀为可检索、可分类、可批量处理的本地素材库。

主要能力：
1. 在 Bilibili 视频页显示悬浮采集面板，一键保存当前视频
2. 自动读取视频基础信息：BV、标题、UP 主、封面、简介、系统标签
3. 支持多收藏夹归档与自定义标签，便于长期整理
4. 与本地管理端联动，统一搜索和管理收藏内容
5. 支持浅色 / 深色 / 自动主题

说明：
- 插件仅在 `bilibili.com/video/*` 页面生效
- 插件通过你本机的本地后端 API 进行数据保存
- 不提供云端账号系统，不主动上传个人数据到开发者服务器
